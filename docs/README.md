## Vue d'ensemble

TreeTracking est une solution de traçabilité forestière basée sur :
- **Hyperledger Fabric 1.4** : Blockchain privée
- **IPFS** : Stockage décentralisé
- **Node.js 12** : API REST
- **React** : Interface utilisateur
